## 🚀 How to Use

1. **Download/Clone Repository**
   ```bash
   git clone https://github.com/sanju-bot/sanjay-portfolio.git
   cd sanjay-portfolio
   ```

2. **View Portfolio**
   - Open `index.html` in a web browser
   - Or use a local server:
     ```bash
     python -m http.server 8000
     # Visit http://localhost:8000
     ```

3. **Customize**
   - Update project details in `index.html`
   - Modify colors in `style.css`
   - Replace `sanjay.png` with your profile picture
   - Update resume file

---

## 🎨 Customization Guide

### Change Colors
Edit the color scheme in `style.css`:
```css
--primary-color: #2d6a4f;    /* Green */
--secondary-color: #a8e6cf;  /* Light Green */
--dark-bg: #0d0d0d;          /* Dark background */
```

### Add More Projects
Duplicate the project card template in `index.html` and update:
- Project title
- Description
- Tech badges
- Links

### Update Skills
Edit the technical skills section in the footer
